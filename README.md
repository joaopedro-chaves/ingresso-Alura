# Simulação de compra de ingressos

Este projeto é um exercício de programação de lógica para simular a compra de ingressos para um evento. O usuário pode selecionar o tipo de ingresso, a quantidade desejada e o sistema calcula o valor total da compra.

Este projeto/exercício é fornecido pela Alura como parte do curso Lógica de Programação.
